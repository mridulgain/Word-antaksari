1. Compile all Class Files:
    -GameServer.java
    -GameClient.java
    -MyStreamSocket.java
    -Dictionary.java

    $ javac GameServer.java MyStreamSocket.java Dictionary.java GameClient.java 

2. Run Server:
    $ java GameServer

3. Run Client:
    $ java GameClient <Server IPAddr>

* No need to provide port No., port no. 2000 is hard coded.
